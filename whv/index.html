新西兰打工度假签证

签证综述

签证概述

飞出国：新西兰打工度假签证，也叫新西兰工作假期签证，英文为 Working Holiday Visa(WHV)。是新西兰工作签证的一种，它允许旅行者出于补贴旅行费用的目的而在签证颁发国边打工边旅行，用来鼓励双方国家的公民进行旅行和文化交流。它允许中国大陆的年轻人在新西兰体验一种不同于家乡的生活，同时不需要通常的提前寻找工作赞助或者参与昂贵的大学交换生计划。

中国大陆年龄18到30岁之间（包含18岁和30岁），并且满足其他条件的申请人可以申请此签证在新西兰打工度假一年，每个人一生只能申请一次。持有签证的申请人可以在新西兰学习或参加培训，累计时间不能超过6个月。新西兰打工度假签证每年开放一次，只能在线申请，并且需要抢配额。WHV签证的有效期为12个月，一定条件下可以延期3个月，最长不超过15个月，之后可以转工签或留学(fcgvisa)。

新西兰是个移民国家，移民具有明确表述，鼓励人们边打工边旅游了解新西兰后再选择定居或学习在新西兰，以减少移民的盲目性，但是你必须符合定居和长期学习的相关条件。 目前有很多人都是通过WHV签证，留在新西兰，尤其以转学生签证的比较多。在转签证的时候，比较简单，只需要自己到新西兰移民局的官方网站下载表格并自行填写，或去移民局在各个新西兰城市的办事处领取表格自行填写并上交材料和费用即可。转为新西兰学生签证的申请过程自己操作，非常简单，并不需要中介的帮助，很多新西兰留学移民中介把申请签证形容为很复杂的一个过程，实际上不是这样的(fcgvisa)。

面向国家

以下42个国家或地区的公民有资格申请新西兰打工度假签证：阿根廷，奥地利，比利时，巴西，加拿大，智利，中国，克罗地亚，捷克共和国，丹麦，爱沙尼亚，芬兰，法国，德国，香港，匈牙利，爱尔兰，以色列，意大利，日本，韩国，拉脱维亚，马来西亚，马耳他，墨西哥，荷兰，挪威，秘鲁，菲律宾，波兰，新加坡（6个月的工作交流项目），斯洛伐克，斯洛文尼亚，西班牙，瑞典，台湾，泰国，土耳其，英国，美国，乌拉圭和越南(flyabroad)。

签证配额

中国大陆：每年1000人，分配方式先到先得，名额用完就要等到下一年(fcgvisa)。
香港地区：每年400人（08年以前是每年200人），分配方式与中国大陆一样。
台湾地区：每年600人（08年以前是每年200人），分配方式：抽签，名额用完一样要等到下一年。

每年开放一次开放时间不固定，之后还会有零星配额开放(fcgvisa)。

  • 2014-2015年度配额开放时间：2014年7月8日；

  • 2016-2017年度配额开放时间：2016年5月23日早上10点，2016年6月23日上午10点会有零星配额开放。

签证有效期

签证有效期为一年，从入境新西兰的第一天开始计算；如果在签证有效期内有效纳税记录超过9个月，则可以延长3个月；最长不超过15个月；签证下来之后一年内必须入境以激活签证，过期不入境新西兰则WHV签证作废(fcgvisa)。

如果超过15个月或未能满足纳税记录长度，但又非常喜欢新西兰，也可以转为其他签证，一般来说转为新西兰学生签证最为简单。

注意事项

  • 一个人一生只能申请一次打工度假签证(flyabroad)；
  • 持有签证后可以在新西兰工作12个月；
  • 持有签证后可以在新西兰学习或参加培训，累计时间不能超过6个月；
  • 打工度假签证只能在线申请；
  • 不能从事永久性工作，也不能为同一个雇主工作超过6个月；
  • 配偶和子女不能随行。

申请条件

  1. 有效护照，计划离境时尚有6个月的有效期(fcgvisa)；
  2. 申请人为中国公民、申请时在中国境内、并且在申请前未有超过两年长期离境；
  3. 年龄在18到30岁之间（包含18岁和30岁），子女不能随行；
  4. 有往返机票，或者提供等额的资金证明（约1W人民币）；
  5. 至少有4200纽币的资金维持在新西兰期间的花销；
  6. 最低高中学历（3年的全日制教育高中学历）。高中学历必须要开具CDGDC证明，高中以上学历只需提供学历原件和翻译件，无需公证；
  7. 满足语言要求，雅思总分5.5分或以上，A类和G类均可，成绩两年内有效。有学习计划的建议考A类(flyabroad)；
  8. 申请费用208新西兰币(flyabroad)；
  9. 以前没有申请过新西兰工作假期签证；
  10. 购买涵盖旅行时间的医疗保险；
  11. 提供体检证明和无犯罪证明。

材料清单

  • 中国护照(fcgvisa)；
  • 胸透或体检证明；
  • 无犯罪证明；
  • 提供往返机票，或者提供等额的资金证明（约1W人民币）；
  • 至少4200纽币的资金证明；
  • 高中学历必须要开具CDGDC证明，高中以上学历只需提供学历原件和翻译件，无需公证；
  • 雅思成绩单（平均分5.5，2年有效期）(flyabroad)；
  • 证明长期居住在中国境内的材料，如寄到你家的信件，或者推荐信等；
  • 证明当前居住在中国的材料，如：水电费账单，或者近期的工资单等；
  • 涵盖旅行时间的医疗保险(flyabroad)。

签证续签

续签概述

飞出国：如果您持有的是新西兰工作假期签证，并且在入境后履行了之前承诺的优先从事季节类的工作，那么满足一定的条件后可以申请延期，最长延期时间为3个月，只允许延期一次。续签后，您的工作假期签证有效期可达15个月。获得延期后要继续从事园艺、葡萄园季节性工作。

续签的前提条件是，在持有 working holiday visa 期间，曾经从事园艺、葡萄园或其他季节性工作至少3个月，续签后将延续原来工作假期签证的签证条款(fcgvisa)。

续签条件

  1. 申请人当前在新西兰境内，且持有工作假期签证 working holiday visa；
  2. 在持有 working holiday visa 期间，曾经从事园艺、葡萄园或其他季节性工作至少3个月(flyabroad)；
  3. 以前没有申请过延期；
  4. 身体健康，无犯罪记录。

注意事项

  1. 工作假期签证只能延期一次；
  2. 季节工作可以不连续，也可以不是为同一个雇主；
  3. 申请延期时无需工作offer(fcgvisa)。

材料清单

  1. 有效护照（计划离境时尚有3个月的有效期）(fcgvisa)；
  2. 2张护照规格的照片；
  3. 延期申请表 SSE/WHE Work Visa Application (INZ 1153) ；
  4. 在持有 working holiday visa 期间，曾经从事园艺、葡萄园或其他季节性工作至少3个月的证明，如雇主推荐信，工资单，税单等；
  5. 有往返机票，或者提供等额的资金证明(flyabroad)；
  6. 无犯罪证明。

常见问题

新西兰假日工作签证能在新西兰停留多久？

签证有效期为一年，从入境新西兰的第一天开始计算；如果在签证有效期内有效纳税记录超过9个月，则可以延长3个月；最长不超过15个月；签证下来之后一年内必须入境以激活签证，过期不入境新西兰则WHV签证作废(fcgvisa)。

一年工作假期签证后我能留在新西兰吗？

首先，刚才提到过，如果在签证有效期内的有效纳税记录超过9个月，可以很容易的延长3个月，最长不能超过15个月。其次，如果超过15个月或未能满足纳税记录长度，但又非常喜欢新西兰，也可以转为其他签证，一般来说转为新西兰学生签证最为简单。

新西兰是个移民国家，移民具有明确表述，鼓励人们边打工边旅游了解新西兰后再选择定居或学习在新西兰，以减少移民的盲目性，但是你必须符合定居和长期学习的相关条件。 目前有很多人都是通过WHV签证，留在新西兰，尤其以转学生签证的比较多。在转签证的时候，比较简单，只需要自己到新西兰移民局的官方网站下载表格并自行填写，或去移民局在各个新西兰城市的办事处领取表格自行填写并上交材料和费用即可。转为新西兰学生签证的申请过程自己操作，非常简单，并不需要中介的帮助，很多新西兰留学移民中介把申请签证形容为很复杂的一个过程，实际上不是这样的。

在新西兰打工需要上税吗？

这个自不用说，新西兰是一个税务制度非常严格的国家，每个持签证的人士都要向新西兰税务局（Inland Revenue）申请新西兰税号（IRD Number），当你找一份工作的要向雇主提供税号， 雇主在发放给你工资的同时，将会按照相应的税率纳税，在新西兰所有的工资都是按照税前收入计算的，例如，每小时15新西兰元的工资，需要纳税20%左右，到手的收入12新西兰元每小时；如果小时工资更高，则税率会有相应的提高，最高可达33%。(flyabroad)

新西兰打工收入如何？生活费用高吗？

新西兰政府规定最低工资是每小时13.75新币，按每周5天8小时计算，到手的收入在450左右。当然你如果肯吃点苦，工作时间再长点，收入就远不止这些。基本生活费为180~250新西兰元每周左右，这其中包括住宿、食物、生活必需品、交通的费用，如果有娱乐的话还会更多些。有些农场季节工包食宿，费用就会省很多。新西兰的生活费用在发达国家中算是比较高的，食物、燃料、公共交通的价格有时会贵的让人难以接受。至于工作招聘的信息，可以去很多大型的超市的留言板、报纸、网站上得到，很多农场也会在收获季节到来前，大量招聘季节工，他们的公司网站会有相应的提示(fcgvisa)。(flyabroad)

新西兰工作假期签证一般做什么类型的工作

原则上讲你只要能找到，你可以从事任何工作，但是为每个雇主工作不能超过3个月。所以多为一些临时性的工作，如每年大量需要的农场季节工，采摘蔬菜水果，为葡萄酒庄摘葡萄，还有餐厅服务，厨房帮手，宾馆客房部，清洁公司等等。当然你如有一技之长，就能找到收入较高的工作。不过实话实说，很多大型的公司不希望招到一个不能稳定工作的人，所以，办公室类型的工作一般不会招聘WHV签证的持有人。

新西兰假日工作签证有英语水平要求吗？

因为你要独自在新西兰工作生活，要求你具备最基本的英语听，说，读，写能力。当然，想申请签证必须获得雅思5.5分以上的成绩，能够得到这个成绩的申请者，基本的英语能力还是可以的。

新西兰假日工作签证有什么福利或津贴吗？

新西兰的假日工作签证没有什么福利或是津贴可以申请。持有人最好在入境的时候就给自己上一份工作签证相关的商业医疗保险，保障自己在生病的时候能够有医疗照顾，新西兰的医疗系统价格昂贵，千万不可以为了省这一点钱不上医疗保险，针对WHV的医疗保障全年下来的价格不过200-300新西兰元而已。另外，只要是入境新西兰的人，如果遇到意外伤害，不用担心医疗费用的问题，新西兰有一个机构叫做ACC，意外和伤害赔偿局，他们对意外事故的赔偿是与签证无关的(fcgvisa)。(flyabroad)

WHV签证好申请吗？具体流程是什么？

申请人可以到新西兰移民局官网查看，有详细的申请流程，直接上新西兰官网自己看，一步一步的点鼠标，得到的都是最新最准确的信息。新西兰WHV签证非常好申请，只要材料充分，雅思成绩合格，只要你能在移民局网站上RELEASE出来名额的时候手够快运气够好，就一定能申请到一个名额。

签证条款

旅行

申请人需要在签证签发后的12个月内入境新西兰。入境后，在签证有效期内，可以自由出入新西兰(fcgvisa)。

入境许可

申请人抵达新西兰后需要申请入境许可。用在飞机上会收到的 arrival card 申请。

如果申请人有以下情况，入境许可可能会被拒绝(flyabroad)：

  • 有犯罪记录；
  • 获签后条件有变化；
  • 拒绝提供照片、指纹或虹膜扫描；
  • 无法提供返程机票或等额的资金证明(flyabroad)。

返程机票

申请人必须提供返程机票或等额的资金证明。

资金证明

提供4200纽币的资金证明(flyabroad)。

医疗保险

购买涵盖旅行时间的医疗保险。

工作

不能有下列情况：

  • 不能为同一个雇主工作超过6个月；
  • 累计工作不能超过12个月；
  • 不能从事长期工作；
  • 工作合法；
  • 不能提供性服务；
  • 不能经营或投资提供性服务的企业(flyabroad)。

注：无需工作offer。

学习
